Version 1.0 Release Notes

Thank you for using Mac Linux USB Loader. I know it's been a while since I've announced the 1.0 release preview, and I know it's been pushed off several times, but I thank you for your patience in waiting for the release. It's been a while, I know, but 1.0 is very much worth it.

You'll find the following changes have occured in the software since my initial beta release:

  • Improved user interface that is stable and not buggy.
  • A distribution downloader has been added to automatically download Ubuntu at the click of a mouse.
  • You can erase your USB drives from the main screen; it is no longer required to open an ISO first.
  • The preferences window has been totally revamped and is now using the excellent RHPreferences framework.
  • It is now easier than ever to create a live USB drive of Linux for Macs!

Roadmap

The future will be a bright one. Here are some things I'm shooting for for future releases:

  • Integration with Enterprise, my custom UEFI boot loader.
  • Removing the current bootx64.efi file and replacing it with my own, custom solution.

That's about it for now. Click the Install & Relaunch button to start using 1.0!
