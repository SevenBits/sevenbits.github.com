Version 3.0.2 Release Notes

Another week, another bug fix. I am pleased to announce version 3.0.2 of Mac Linux USB Loader.

This release fixes two issues which have been plaguing certain users of Mac Linux USB Loader:

  • Issue #63: Fixes an issue whereby a configuration file would not be created on the specified USB drive
  • Issue #62: Fixes an issue whereby Enterprise could not be found, aborting the installation

This release also contains a number of important improvements to the installation wizard, making it a little bit easier to use. Specifically, the Back and Forward are now enabled and disabled based off of what screen you are on, etc. The program also prints more log messages to aid finding more bugs in the future.

Well, until next time! Please click Learn More... to be directed to my webpage where you can download the new version.

Version 3.0 Release Notes

I am pleased to announce to world that the Mac Linux USB Loader rewrite is officially ready to be released as version 3.0! It features brand new, never-before-seen features in any live USB creator for Mac, such as:

  • An improved Enterprise boot manager, containing fixes for many common errors, including the dreaded "booting in blind mode"
  • A brand-new user interface, which is much easier to use and more Mac-like
  • Persistence support for Ubuntu-based distributions: setup persistence on your USB drive to save data between runs
  • An improved distribution downloader, with a list of mirrors that automatically updates from the web
  • Added experimental support for USB drives formatted as HFS+
  • Added the ability to list Mac Linux USB Loader-created USB drives in OS X's startup disk selector
  • You can register multiple copies of Enterprise with Mac Linux USB Loader and choose which version you would like to use when creating a new USB
  • And more!

Because it has been entirely reprogrammed from scratch, you cannot automatically update to the new version from this version. You will need to download the update manually by clicking on the Learn More... button below.

That's about it. Thank you for your patience.

Version 2.0 Release Notes

Mac Linux USB Loader version 2.0 is a groundbreaking release, shipping with the revolutionary new boot loader Enterprise, making it even easier to make Linux boot from USB drives on Macs. Even better, it requires zero configuration! Just load up Mac Linux USB Loader and pop in a supported distribution and you're good to go!

Please note that you may need to download the update manually. If you're upgrading from the version 2.0 beta, you should be fine.

Version 1.1 Release Notes

Mac Linux USB Loader version 1.1 is mainly a bug fix release. Thanks to the many users who have taken the time to point out and report bugs, as well as suggest ideas for new features.

Here's a list of changes that are in this release:

  • Fixed a issue where Mac Linux USB Loader would crash after downloading a Linux distribution from the Download Distribution panel.
  • Fixed some issues regarding the progress bar in the installation window incorrectly reporting the file copy progress.
  • You can now close Mac Linux USB Loader's main window; clicking the Dock icon will bring it back.
  • Added an option on the main menu screen to "bless" a USB drive created with Mac Linux USB Loader, allowing to boot up automatically.
  • Added a Firmware Options section in the Preferences window, allowing you to customize options allowing you to choose which firmware to install.
  • Updated Spanish localizations of user interface elements.

My custom bootloader, Enterprise, did not make it in time for this release. I will be issuing an update when it is ready. There is an option in the Firmware Options panel to set Enterprise as the preferred bootloader, so you'll have your preferences all set when the final version does ship!

Roadmap

The future will be a bright one. Here are some things I'm shooting for for future releases:

  • Integration with Enterprise, my custom UEFI boot loader.
  • Removing the current bootx64.efi file and replacing it with my own, custom solution.

That's about it for now. Click the Install and Relaunch button to start using Mac Linux USB Loader version 1.1!
